
# Watch Together Api

A tiny Socket api for Watch-Together app

## Tech Stack
```
- Express
- Socket.io
```

Demo : https://wtg.batusan.dev


## Run Locally

Clone the project

```bash
  git clone https://github.com/batusan/socket-api
```

Go to the project directory

```bash
  cd socket-api
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn run dev
```
